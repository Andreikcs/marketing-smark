# Swipe File — referências de forma

Catálogo de referências que guiam **forma** (hook, formato, visual). Princípio: **roubar a forma, rejeitar a substância** ([[shared/padroes-instagram]] seção 5). Os comandos consultam este arquivo por tags.

## Energia de hook (BR, alto engajamento) — roubar forma, rejeitar tom

### @v4company
- **Extrair:** conteúdo de autoridade ("POR QUE CONFIAR / POR QUE CONTRATAR"), hook de 3 palavras, alto contraste.
- **Evitar:** vermelho gritante, métrica inflada (+8.5 BI), hype.
- **Tags:** `tom-direto` `hook-afirmacao-polemica` `formato-por-que-confiar` `obj-awareness`

### @g4.business
- **Extrair:** storytelling/analogia (Blockbuster, Santos Dumont), quote card, ritmo cinematográfico, retrato dramático.
- **Evitar:** provocação que insulta o gestor, número de aspiração vazio ("10 milhões/ano").
- **Tags:** `storytelling-analogia` `tom-narrativo` `visual-foto-real` `obj-awareness`

### @weagleai (Pedro Muschitz)
- **Extrair:** hook + subhook entre parênteses, how-to de IA, listicle.
- **Evitar:** imagem AI cafona, "a IA deixa as pessoas burras", tom de salvador.
- **Tags:** `tom-direto` `hook-afirmacao-polemica` `obj-educacao` `newsjacking`
